Allows admins to disable the Visual editor tab on a per post basis.

== Description ==
Plugin to disable widgets in the edit view of Posts/Pages.

Code from these other plugins has served as reference and inspiration, and was sometimes borrowed from:

https://wordpress.org/plugins/dont-break-the-code/

== Installation ==

1. Upload the `content-editor-unbounder` directory to the `/wp-content/plugins/` 
directory of your site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Look for the checkboxes in the Screen Options of the edit post page under the headline 'Content Editor Unbounder'.
